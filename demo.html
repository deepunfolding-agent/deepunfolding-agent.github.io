This interactive demo illustrates how the DeepUnfolding Agent API performs autonomous optimisation control: receiving a goal, creating a controller, suggesting bounded controls, executing a round, receiving feedback, updating the controller, and deciding whether to continue or stop.

Autonomous Round Loop

Observe → decide → act → evaluate → update

Goal received

POST /agent/run-autonomous-experiment

The API receives the objective, diagnostics, constraints, and executor type.

Schema loaded

GET /strategies

The agent retrieves the expected diagnostics and control-variable bounds.

Controller created

POST /controllers/create

The controller backend is instantiated for the selected strategy.

Controls suggested

POST /controllers/{id}/suggest-controls

The deep-unfolded controller proposes bounded control variables.

Round executed

POST /quantum/run-controlled-round

The executor applies the controls and returns feedback metrics.

Controller updated

POST /controllers/{id}/update-controller

The controller uses feedback to improve future control suggestions.

Decision made

continue / stop / save best

The evaluator decides whether to continue, stop, or save the best run.
