Strategy-Agnostic Adaptive Controller API

Deep-Unfolding Agent

DeepUnfolding Agent is a FastAPI-based agent framework that receives runtime diagnostics from an external optimization system and returns bounded strategy-level control variables. The framework supports classical learning, quantum learning, and hybrid classical-quantum learning optimization through a common schema-driven interface.

Core Idea

Adapt the strategy, not only the model optimiser

Existing adaptive optimisers usually modify parameter updates inside a model optimiser. DeepUnfolding Agent instead operates at the strategy level. It observes diagnostics from an external system and returns control variables that can shape the next optimisation round.

External API Boundary

Training systems remain independent. They send diagnostics to the controller service and receive bounded control suggestions through a stable HTTP interface.

Schema-Driven Control

Each strategy defines ordered state variables and bounded control variables. This allows user-defined strategies to be registered without rewriting the controller API.

Deep-Unfolded Backend

Learnable multi-step controller logic adapts strategy-level variables across repeated optimisation rounds using feedback from the external execution loop.

Controller Families

One interface for multiple optimisation settings

C

Classical

Controls learning rate, proximal strength, local epochs, and other strategy-level variables in classical and federated learning systems.

Q

Quantum

Controls quantum parameter learning rate, shot count, circuit depth, entanglement depth, and measurement reuse.

H

Hybrid

Coordinates classical and quantum controls through a shared schema-driven controller interface.

Public Interface

API endpoints

GET/health

Check whether the controller service is running.

GET/strategies

List built-in and user-registered strategy schemas.

POST/strategies/register

Register a user-defined optimisation strategy schema.

POST/controllers/create

Create a controller instance for a selected strategy.

POST/controllers/{id}/suggest-controls

Return bounded controls from the latest diagnostics.

POST/controllers/{id}/update-controller

Update the controller using feedback from the external round.

POST/quantum/run-controlled-round

Execute a controlled quantum round using registered controls.

GET/experiments/{run_id}

Retrieve logged experiment outputs and metadata.

Quickstart

Run locally and open Swagger UI

Start the FastAPI service locally, then use Swagger UI to create a controller, submit diagnostics, request controls, and update the controller after each external optimisation round.
